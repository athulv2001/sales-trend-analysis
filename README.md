# Sales Trend Analysis

This project analyzes weekly sales data to identify demand patterns, growth trends, and performance fluctuations.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Steps
1. Data exploration
2. Calculation of average sales
3. Detection of weekly sales changes
4. Trend classification (Growth / Decline / Stable)
5. Visualization of weekly sales trends

## Key Insights
- Average weekly sales were approximately **245 units**
- Week **8** recorded the highest sales
- Weeks **4 and 7** experienced the largest growth
- Small declines occurred in weeks **3 and 6**

## Visualization
The project includes a line chart showing the weekly sales trend.
